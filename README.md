PROJECT DESCRIPTIONS:
An AI-driven plant disease detection platform that empowers farmers with instant diagnosis and multilingual voice-assisted treatment guidance for smarter and more accessible crop care.
FEATURES:
1. Leaf image disease detection.
2. Real-time camera detection.
3. Treatment suggestions.
4. Weather-based prediction.
5. Local Agricultural Supply Stores.
6. Multilanguage and helpline
TECHNICAL STACK:
Frontend: Flask + Jinja2 + HTML/CSS/JS
AI/ML: OpenCV + Pre-trained CNN
Speech: gTTS + Pygame
Maps: Google Maps API
Weather: OpenWeatherMap
Deployment: Railway/Heroku
